# ISB | ISuite Balance

![ISB Illustration](ISuiteBalance/src/main/resources/static/img/conta-illustration.png)

##### "Tu negocio, tus cuentas, tu orden."
------------

### ¿En qué consiste?

- Es un programa que toda empresa debería implementar para la gestión de su contabilidad. Integra un sistema de roles estructurado para que cada persona vea únicamente lo que le corresponde. El Auditor no tiene por qué andar de curioso revisando lo que no debe.

- Gestión de usuarios con control total para el Administrador. Puede asignar roles, retirarlos y administrar accesos sin complicaciones. Si tienes a alguien que se mete donde no debe, simplemente eliminas su cuenta y asunto arreglado.

- Inserción de partidas con soporte para múltiples cuentas afectadas, respaldadas por un archivo PDF por cada registro. No más partidas dudosas ni cuentas inventadas. Puedes añadir varias líneas con un clic y corregir errores fácilmente mediante la opción de eliminar líneas.

- Incluye todo el proceso contable básico: registro, clasificación, consulta, verificación y auditoría. El sistema está diseñado para mantener la información clara, ordenada y accesible, sin pasos innecesarios ni pantallas confusas.

- Una interfaz sencilla, directa y pensada para trabajar rápido. Navegación intuitiva mediante un sidebar que organiza todos los módulos del sistema.

### ¿Qué incluye ISuite Balance?

El sistema cuenta con los siguientes módulos principales:

#### **Gestión de roles y usuarios**
- Asignación de roles: Administrador, Contador y Auditor.
- Control total de accesos según el rol.
- CRUD completo de usuarios (crear, leer, actualizar y eliminar).

#### **Gestión de partidas**
- Registro de partidas contables con soporte para múltiples cuentas afectadas.
- Carga de archivos PDF como respaldo documental.
- Validación automática del Debe y el Haber.
- Edición y eliminación de partidas.
- Vista detallada de cada movimiento y su documento asociado.

#### **Libro Diario**
- Registro cronológico de todas las partidas.
- Visualización clara por fecha y concepto.
- Base para auditorías y análisis contable.

#### **Libro Mayor**
- Consolidación automática de movimientos por cuenta.
- Cálculo de saldos final y acumulado.
- Facilita el análisis del comportamiento de cada cuenta.

#### **Balanza de Comprobación**
- Comparación entre la suma total del Debe y del Haber.
- Verificación del equilibrio contable previo a informes.

#### **Auditoría y Bitácora**
- Registro de todas las acciones realizadas por los usuarios.
- Historial claro de modificaciones, accesos y operaciones.
- Herramienta para control interno y transparencia.

#### **Reportes**
- Generación de reportes listos para imprimir.
- Exportación en PDF.
- Resumen claro de métricas del sistema.
